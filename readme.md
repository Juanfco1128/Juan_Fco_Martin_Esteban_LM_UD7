Práctica UD7 - Conversión y adaptación de documentos
Temática elegida

La temática elegida ha sido películas, ya que permite trabajar con datos estructurados y jerárquicos como título, año, género, director, duración y puntuación.

Partes realizadas
Conversión XML - JSON

Se ha utilizado Python y la librería xmltodict para convertir los datos entre formato XML y JSON manteniendo la estructura original.

Transformación XML - HTML con XSLT

Se ha creado una plantilla XSLT que transforma el archivo XML en una página HTML con una tabla que muestra la información de las películas.

Generación HTML desde JSON con JavaScript

Se ha desarrollado una página web que carga un archivo JSON mediante JavaScript y muestra dinámicamente la información de las películas utilizando HTML y CSS.
